# Hydro Pastebin Plus

由 `hydro_pastebin` 二次开发。

## 使用说明

1. 进入您的 HydroOJ 安装目录
2. 执行如下命令
   ```bash
   yarn add hydrooj-pastebin-plus
   hydrooj addons add hydrooj-pastebin-plus
   ```
3. 重启您的 HydroOJ
4. 在首页菜单加入超链接至 `/paste/create`
